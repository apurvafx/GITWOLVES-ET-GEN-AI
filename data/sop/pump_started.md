# Standard Operating Procedure

## Startup Procedure

Equipment:
PUMP-101A

### Before Startup

- Verify lubrication.
- Verify pump alignment.
- Ensure suction line is full.
- Open suction valve.
- Partially open discharge valve.
- Verify electrical connections.
- Ensure coupling guard is installed.

### Start

1. Start motor.
2. Check rotation.
3. Observe vibration.
4. Observe bearing temperature.
5. Observe discharge pressure.
6. Gradually open discharge valve.

### During Operation

Monitor

- vibration
- bearing temperature
- abnormal noise
- leakage
- motor load

If abnormal vibration occurs

Stop pump immediately.
